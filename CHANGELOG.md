# Changelog

## 0.1.0 - 2026-06-26

### Added

- Add configurable CV templates with `default` preserving the existing layout.
- Add configurable color themes with `catppuccin-latte`, `default`, and `graphite`.
- Add a standalone shorter homepage that links to the GitHub repository.
- Pin Ruby with `mise` for local builds.

### Changed

- Make Catppuccin Latte the default theme.
- Upgrade Jekyll, jQuery, Moment.js, and flag-icons.
- Improve the print/save banner contrast in the Catppuccin Latte theme.

### Fixed

- Fix the Moment.js SRI hash so age and experience calculations run in browsers.
